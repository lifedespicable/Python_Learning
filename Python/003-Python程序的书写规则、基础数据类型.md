- 如果是有一串数字或者是一串字符，没有办法区分它们类型的话，在这里有一个工具叫做type()，以此来判断它是4种类型中的哪一种类型

  - 把数字转化为布尔值bool类型，bool(123)，只要数字不为0就是True，只要数字为0就为False
